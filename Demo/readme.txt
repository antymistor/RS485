	  为了适应不同的控制模式，机械臂的程序一共提供6种控制模式：这里介绍一下。下位机可以通过上位机的指令参数范围来判断接下来应该工作在哪种模式下。
      单条指令的指令形式为：X0X1X2X3X4 ，单条指令由5个字节组成。

      Mode 0:停机模式
      指令参数范围： X0=A-F, X1=S,X2X3X4=无限制。
      控制形式：所有电机停转
      例：AS123，进入停机模式

      Mode 1:单关节位置控制
      指令参数范围： X0=A-F, X1X2X3X4=0-9
      控制形式： 该模式用于将指定关节旋转至指定角度
                 X0指定控制关节号，A-F分别指定1-6号关节。
                关节位置转角= X1*100+X2*10+X3+X4*0.1-180
      例：B1523,将2号关节旋转至-27.7deg度的位置

      Mode 2:单关节短步控制
      指令参数范围：X0=A-F, X1=m-o, X1X2X3X4=0-9
      控制形式： 该模式用于将指定关节在当前位置的基础上旋转一小步
                 X0指定控制关节号，A-F分别指定1-6号关节。
                 X1指定旋转方向，m正旋，n反转，o停转
                 旋转时间= （X2*100+X3*10）ms
                 旋转速度=  X4*100（油门）
      例：Cn547,将3号关节反方向以700的‘速度’旋转540ms.
      

      Mode 3：多关节同步位置控制
      指令参数范围： X0=a-f, X1X2X3X4=0-9
      控制形式： 该模式用于将所有关节旋转至指定角度
                 X0指定控制关节号，a-f分别指定1-6号关节。
                 关节位置转角= X1*100+X2*10+X3+X4*0.1-180
      例：a1870b1730c2250d1400e2002f1770,将1—6号关节分别旋转至+7.0deg,-7.0deg,+45.0deg,-40deg,+20.2deg,-3.0deg

      Mode 4：XYZ-noa 笛卡尔坐标系位置控制
      指令参数范围： X0=g-l, X1X2X3X4=0-9
      控制形式： 该模式用于将机械臂末端移动到到指定笛卡尔空间坐标位置，以及指定位姿
                  X0指定不同的坐标方向。g,h,i分别指代X,Y,Z坐标方向。


                                        j,k,l分别指代n-o-a三个欧拉姿态角
                 末端位置（cm）/姿态(deg)= X1*100+X2*10+X3+X4*0.1-180
       例：g1870h1730i2250j1400k2002l1770,将机械臂的末端执行器移动到坐标为（+7.0cm,-7.0cm,-45.0cm）的位置，并使得机械臂末端姿态角为（-40deg,20.2deg,-3.0deg）。
      
      Mode 5: XYZ笛卡尔坐标系位置短步控制
      指令参数范围： X0=x-z, X1X2X3X4=0-9
      控制形式： 该模式用于将机械臂末端在笛卡尔坐标系内延坐标轴方向移动指定距离
                 X0指定不同的坐标方向。x,y,z分别指代X,Y,Z坐标方向。
                 末端位移量（mm）= X1*100+X2*10+X3+X4*0.1-180
       例：z2029,将机械臂末端延Z轴正方向移动22.9mm。
